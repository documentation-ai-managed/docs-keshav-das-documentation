<CodeGroup tabs="JavaScript,Python">
  ```javascript
const vaibrix = require('vaibrix-sdk');
const client = new vaibrix.Client({ apiKey: 'your-api-key' });
const chatbot = await client.createChatbot({
  name: 'Support Bot',
  personality: 'helpful'
});
console.log('Chatbot created:', chatbot.id);
```
  ```python
from vaibrix_sdk import VaibrixClient
client = VaibrixClient(api_key='your-api-key')
chatbot = client.create_chatbot(
    name='Support Bot',
    personality='helpful'
)
print(f'Chatbot created: {chatbot.id}')
```
</CodeGroup>
